# SmashDiscordBot
Discord Bot written in Python, start to learning Python.

Bot meant to return frame data information about characters in Super Smash Bros. Ultimate, with frame data from Ultimate Frame Data, and other associated databases online. 
https://ultimateframedata.com/

Also will return Spotify and youtube playlists for music from the specified character's series of game. For example, !Joker would return information about Joker in Super Smash Bros. Ultimate, and would return a playlist from spotify and youtube from his series of games, Persona 5.
